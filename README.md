# 🌯 Telegram-бот для донерной

Бот принимает заказы, собирает корзину и хранит отзывы клиентов.

---

## 📂 Структура файлов

```
doner_bot/
├── bot.py           — точка входа, запуск бота
├── config.py        — токен и настройки
├── handlers.py      — вся логика обработки команд и кнопок
├── keyboards.py     — кнопки и клавиатуры
├── database.py      — работа с SQLite
├── states.py        — состояния FSM (машина состояний)
└── requirements.txt — зависимости
```

---

## 🚀 Инструкция по запуску (шаг за шагом)

### Шаг 1 — Создайте бота в Telegram

1. Откройте Telegram и найдите **@BotFather**
2. Напишите `/newbot`
3. Придумайте **имя** бота (например: `Донерная ДонерХаус`)
4. Придумайте **username** бота — должен заканчиваться на `bot`  
   (например: `donerhaus_bot`)
5. BotFather пришлёт вам **токен** вида:  
   `7123456789:AAFxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`
6. **Сохраните токен — он понадобится на следующем шаге**

---

### Шаг 2 — Установите Python

Убедитесь, что Python версии **3.10 или выше** установлен:

```bash
python --version
# или
python3 --version
```

Если не установлен — скачайте с [python.org](https://python.org/downloads)

---

### Шаг 3 — Скачайте файлы бота

Скопируйте все файлы из архива в одну папку, например `doner_bot/`

---

### Шаг 4 — Создайте виртуальное окружение

Это изолирует зависимости от других проектов.

**Windows:**
```cmd
cd doner_bot
python -m venv venv
venv\Scripts\activate
```

**macOS / Linux:**
```bash
cd doner_bot
python3 -m venv venv
source venv/bin/activate
```

После активации вы увидите `(venv)` в начале строки терминала.

---

### Шаг 5 — Установите зависимости

```bash
pip install -r requirements.txt
```

Дождитесь завершения установки.

---

### Шаг 6 — Вставьте токен в конфиг

Откройте файл **`config.py`** в любом редакторе (Notepad, VS Code и т.д.)

Найдите строку:
```python
BOT_TOKEN = os.getenv("BOT_TOKEN", "ВАШ_ТОКЕН_ЗДЕСЬ")
```

Замените `ВАШ_ТОКЕН_ЗДЕСЬ` на ваш токен от BotFather:
```python
BOT_TOKEN = os.getenv("BOT_TOKEN", "7123456789:AAFxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx")
```

Также можете изменить:
- `RESTAURANT_NAME` — название вашей донерной
- `CONTACT_INFO` — ваш номер телефона

---

### Шаг 7 — Запустите бота

```bash
python bot.py
```

Вы должны увидеть в консоли:
```
2024-01-15 12:00:00 [INFO] root: База данных инициализирована
2024-01-15 12:00:00 [INFO] root: Бот запущен. Нажмите Ctrl+C для остановки.
```

Откройте Telegram, найдите вашего бота по username и напишите `/start` 🎉

## ⚙️ Настройки и расширение

### Изменить цены или добавить блюда

Откройте `keyboards.py`, найдите список `MENU_ITEMS`:

```python
MENU_ITEMS = [
    ("🌯 Донер с курицей",   "250", "item_doner_chicken"),
    ("🥩 Донер с говядиной", "300", "item_doner_beef"),
    ("🍟 Картошка фри",      "120", "item_fries"),
    ("🍗 Наггетсы",          "150", "item_nuggets"),
]
```

Каждая строка: `("Название", "Цена", "уникальный_ключ")`

Добавьте новую строку по тому же образцу. Затем добавьте описание блюда в `handlers.py` в словарь `descriptions`.

### Где хранятся данные?

После запуска появится файл **`doner_bot.db`** — это база данных SQLite.  
Открыть и посмотреть можно через бесплатную программу [DB Browser for SQLite](https://sqlitebrowser.org/)

## 🖥 Запуск как фоновый сервис (Linux/VPS)

Чтобы бот работал даже после закрытия терминала, используйте `systemd` или `screen`:

```bash
# С помощью screen
screen -S doner_bot
python bot.py
# Нажмите Ctrl+A, затем D — выйти из screen, бот продолжит работать

# Вернуться к боту:
screen -r doner_bot
```

## ❓ Часто задаваемые вопросы

**Q: Бот не отвечает**  
A: Проверьте что токен вставлен правильно и бот запущен (видно в консоли)

**Q: Ошибка `ModuleNotFoundError`**  
A: Убедитесь что виртуальное окружение активировано и зависимости установлены

**Q: Как узнать ID пользователя / чата**  
A: Напишите боту `@userinfobot` в Telegram
